# Facebook Ads Tutorial

## Muc tieu
- Dong bo chi so Facebook Ads vao Notion de theo doi theo ngay/tuan.

## Quy trinh nhanh
1. Mo tab HTTP Request.
2. Chon `GET`.
3. Dan URL Graph API Ads endpoint.
4. Kiem tra `Parameters` va bo sung `date_preset`, `fields`.
5. Bam `Run` de xem response.
6. Kiem tra `Raw Response` va keo-tha field map vao body/header/parameter neu can.

## Goi y field thuong dung
- spend
- impressions
- clicks
- cpc
- cpm
- ctr

## Luu y
- Nen su dung account token co quyen read ads insight.
- Neu response lon, su dung filter field de giam payload.
