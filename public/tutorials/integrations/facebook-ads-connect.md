# Facebook Ads Connect Guide

## Cach ket noi
1. Nhan `Connect` tren card Facebook Ads.
2. Neu popup OAuth mo ra, dang nhap tai khoan Meta.
3. Cap quyen doc Ads data.
4. Quay lai console va chay lai request.

## Neu gap loi
- Kiem tra endpoint `/api/providers/facebook/connect` co duoc bat hay khong.
- Neu ban dang o ban build demo, OAuth co the bi tat tam thoi.

## Thiet lap thu cong
- Chuyen qua `Token` mode.
- Dan Bearer token vao Authorization.
- Thu lai request.
