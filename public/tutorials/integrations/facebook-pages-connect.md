# Facebook Pages Connect Guide

## Ket noi nhanh
1. Nhan `Connect`.
2. Hoan tat OAuth voi quyen doc page insights.
3. Kiem tra lai request URL va page id.

## Fallback
Neu OAuth chua available trong build hien tai:
- Dung `Token` mode voi access token hop le.
- Them `Authorization: Bearer <token>` trong tab Authorization.
