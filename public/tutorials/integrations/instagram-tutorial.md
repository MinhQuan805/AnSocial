# Instagram Tutorial

## Muc tieu
Trich xuat account insights va media insights theo ngay.

## Buoc lam
1. Chon URL Graph API cho Instagram.
2. Nhap parameters: metric, period, since, until.
3. Chay request trong HTTP mode hoac OAuth mode.
4. Kiem tra raw json va map field keo-tha.

## Metrics thuong dung
- reach
- impressions
- accounts_engaged
- profile_views
